# Diabetes Prediction Model

## Live Demo
Check out the live application here: [Diabetes Risk Predictor](https://diabetes-risk-predictor-2279.onrender.com)

## Overview
Developed a machine learning-based system to predict the likelihood of diabetes in patients using clinical and demographic data. The model focuses on handling imbalanced datasets and improving early detection performance.

---

## Objectives
- Predict diabetes occurrence with high recall for better medical screening
- Compare multiple machine learning models for performance optimization
- Handle class imbalance effectively

---

## Models Used
- Random Forest  
- XGBoost  
- Naive Bayes  

---

## Dataset
- Over **900,000 patient records**
- Includes medical attributes such as glucose levels, BMI, age, and other health indicators

---

## Methodology
- Performed **data preprocessing** including missing value handling and feature scaling  
- Conducted **univariate, bivariate, and trivariate analysis** for feature insights  
- Applied **one-hot encoding** and feature standardization  
- Addressed class imbalance using **resampling techniques**  
- Trained multiple models and compared performance  

---

## Results
- Achieved **94% test accuracy**
- Attained **80% recall** on imbalanced data  
- Built a robust end-to-end ML pipeline for prediction  

---

## Tech Stack
- **Languages:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, XGBoost  
- **Tools:** Jupyter Notebook, Google Colab
